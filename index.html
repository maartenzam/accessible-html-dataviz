HTML 101 and accessible HTML

This page contains links to web pages that illustrate basic and accessibility related features of HTML. These pages were developed for the Accessibility and data visualisation training.

For each of the listed pages, a link to the live page is given (to be rendered with a browser and read by a screen reader) as well as a link to the raw HTML code.
